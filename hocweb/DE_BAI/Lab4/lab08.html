Bài 08: JavaScript - Phương thức DOM HTML

Tham khảo mã nguồn:

    <div class="row">
        <div class="col-sm-4">
            <h5 id="h5_tieuDe"> </h5>
            <hr>
            <label for="txt_hoTen" id="lb_hoTen"></label>
            <input type="text" id="txt_hoTen" name="txt_hoTen" class="form-control">
            <label for="txt_email" id="lb_email"></label>
            <input type="text" id="txt_email" name="txt_email" class="form-control" >
        </div>
    </div>    
    <script type="text/javascript">
        // gán text:
        document.getElementById("h5_tieuDe").innerText = "Ví dụ về: document.getElementById:";
        // gán style:
        document.getElementById("h5_tieuDe").style.color = "Red";
        // gán text cho thẻ lable:
        document.getElementById("lb_hoTen").innerHTML = "Họ và tên:";
        // gán textcho input text:
        document.getElementById("txt_hoTen").value = "Hoàng Việt Anh";
        // gán text cho thẻ lable:
        document.getElementById("lb_email").innerText = "Email:";
        // gán placeholder cho input text:
        document.getElementById("txt_email").placeholder = "Nhập Email..."
    </script>

Mở Developer Tools bằng F12 (hoặc click vào dấu 3 chấm phía trên bên phải => More Tools => Developer Tools).
